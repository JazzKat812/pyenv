pyenv commands
==============

Lists all available pyenv commands.


Synopsis
--------

.. code-block:: shell
    
    pyenv commands [--sh|--no-sh]


Options
-------

.. program:: pyenv commands

.. option:: --sh, --no-sh

    description TODO


Description
-----------

The command :command:`pyenv commands` lists all available subcommands
of pyenv.

.. code-block:: shell-session

    $ pyenv commands
    commands
    completions
    exec
    global
    help
    [...]
